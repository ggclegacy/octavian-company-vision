# OCTAVIAN SESSION TEST CHECKLIST

1. Run `npm install`.
2. Run `npm run dev`.
3. Open the app.
4. Start the session.
5. Answer at least one question in every category.
6. Test typing an answer.
7. Test voice input if browser supports it.
8. Test Skip for Now.
9. Test Needs Follow-Up.
10. Refresh the page and confirm answers restore.
11. Generate / update the vision draft.
12. Review the vision together.
13. Add feedback to at least two sections.
14. Finalize the Vision Hub.
15. Open the Vision Hub.
16. Confirm the Future Build Roadmap appears.
17. Open Export.
18. Copy each output.
19. Download full session JSON backup.
20. Clear session only after confirming backup works.
21. Import the backup and confirm session restores.
